## 📖 Overview

This repository contains the source code for my professional portfolio website. The site serves as a centralized hub to present my technical capabilities, host live demonstrations of my artificial intelligence projects, and provide an ATS-optimized, mathematically aligned digital resume.

My core focus lies in **Generative AI**, **Retrieval-Augmented Generation (RAG)**, **Explainable AI (XAI)**, and building end-to-end intelligent systems.

## 🛠️ Architecture & Tech Stack

The architecture of this portfolio prioritizes performance, accessibility, and clean code, explicitly avoiding heavy framework bloat.

- **Frontend Core:** Pure HTML5, Vanilla JavaScript.
- **Styling Engine:** Custom CSS3 leveraging CSS Grid and Flexbox for native responsiveness across mobile, tablet, and desktop viewports.
- **UI/UX:** Implements a premium dark-mode aesthetic with "glassmorphism" components and hardware-accelerated micro-animations.
- **PDF Generation Pipeline:** The resume (`resume.html`) is highly engineered with `@media print` CSS rules, allowing it to natively scale and render as a perfect one-page A4 PDF without any manual print configuration.
- **Hosting:** Deployed via GitHub Pages for global edge-network delivery.

## 🔬 Featured AI/ML Projects

This portfolio highlights my direct contributions to the following engineering projects:

### 1. Multimodal Fake News Detection System (RAG & XAI)
- **Architecture:** Hybrid Cloud AI utilizing open-source (RoBERTa, TextBlob) and Cloud APIs (Gemini 3.5 Flash, Groq Llama-3.1).
- **Core Features:** RAG pipeline for live web fact-checking via Serper API, integrated with XGBoost for Explainable AI.
- **Deployment:** Scalable backend deployed via FastAPI and Docker on Hugging Face Spaces.

### 2. Autonomous AI Research Agent
- **Architecture:** High-speed GenAI agent leveraging Groq (Llama 3), FAISS vector databases, and Sentence Transformers.
- **Core Features:** Automated PDF report synthesis, advanced hallucination detection, and real-time web scraping algorithms.
- **Deployment:** Interactive user interface deployed on Streamlit Cloud.

### 3. Employee Salary Prediction
- **Architecture:** End-to-end Machine Learning pipeline using XGBoost and SMOTE data balancing.
- **Core Features:** SHAP Explainable AI (XAI) integration to interpret model decisions and feature importance.
- **Deployment:** Interactive data dashboard deployed via Streamlit.

## 📥 Local Development

To clone and run this portfolio locally for development or auditing purposes:

```bash
# Clone this repository
git clone https://github.com/jagannadharao8/jagannadharao8.github.io.git

# Navigate into the directory
cd jagannadharao8.github.io

# Launch a local server (requires Python)
python -m http.server 8000
```
Then navigate your browser to `http://localhost:8000`.
