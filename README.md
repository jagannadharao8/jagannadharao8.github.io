Welcome to the source code for my personal AI & Machine Learning engineering portfolio.

## 🚀 About This Portfolio
This repository hosts my professional web portfolio, designed to showcase my projects in **Generative AI**, **RAG (Retrieval-Augmented Generation)**, **Explainable AI (XAI)**, and predictive machine learning architectures. 

The site serves as a central hub for recruiters and developers to access my live project demos (hosted on Hugging Face and Streamlit), review my technical skills, and download my ATS-optimized professional resume.

## 🛠️ Tech Stack & Features
- **Frontend Core:** Pure HTML5, Vanilla JavaScript, and heavily optimized CSS3 (Grid/Flexbox).
- **Styling:** Premium dark-mode UI elements with subtle micro-animations and "glassmorphism" card designs to create a dynamic, modern feel without the bloat of heavy frameworks.
- **Responsive Design:** 100% mobile-responsive layout for seamless viewing on mobile, tablet, and desktop.
- **ATS-Optimized Resume Engine:** Includes a dedicated CSS print-media queried resume (`resume.html`) that algorithmically structures the DOM to export a mathematically perfect, one-page A4 PDF without scale configuration.

## 📂 Key Projects Featured
1. **Multimodal Fake News Detection System (RAG & XAI):** A Hybrid Cloud AI utilizing Gemini and Llama-3.1 with Explainable AI integration via XGBoost.
2. **Autonomous AI Research Agent:** A high-speed agent using Groq and FAISS for automated PDF synthesis and semantic search.
3. **Employee Salary Prediction:** An end-to-end ML pipeline featuring SHAP visualizations deployed via Streamlit.
